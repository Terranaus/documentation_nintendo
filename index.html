Documentation personnel sur Nintendo

Bienvenue dans une documentation française sur Nintendo !

Cette documentation non-officielle couvre les différentes consoles Nintendo, leurs spécifications techniques, et leurs caractéristiques uniques.

Utilisez le bouton en haut à droite pour basculer entre les thèmes clair et sombre.

Dernières consoles

Nintendo 3DS

Console portable avec écran 3D. Lancée en 2011.

Spécifications :
- Processeur : Dual-Core ARM11
- RAM : 128MB
- Stockage : 1GB interne

Nintendo Switch

Console hybride. Lancée en 2017.

Spécifications :
- Processeur : NVIDIA Tegra
- RAM : 4GB
- Stockage : 32GB (extensible)

Nintendo Switch 2

Successeur de la Switch. Lancée en 2025.

Spécifications :
- Processeur : NVIDIA Tegra T239
- RAM : 8GB ou 12GB
- Stockage : 128GB ou 256GB

Comment utiliser cette documentation

Utilisez la barre latérale à gauche pour naviguer entre les différentes sections et consoles. Les liens en surbrillance indiquent la page actuelle.
